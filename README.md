### create a new repository 

```markdown
echo "# junlanblog" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/LIJIEJIE519/junlanblog.git
git push -u origin main
```

```markdown
git remote add origin https://github.com/LIJIEJIE519/junlanblog.git
git branch -M main
git push -u origin main
```
